# 🏍️ Okada Online — Web App

## Complete 4-App Transport Ecosystem for Ghana

**Passenger · Driver · Owner · Admin**

### Revenue Splits
- 70% → Vehicle Owner (auto-paid)
- 10% → Driver (auto-paid)
- 3%  → Fuel Pool (locked)
- 2%  → Maintenance Pool (locked)
- 15% → Platform

### Quick Deploy to Vercel
1. Push this folder to GitHub
2. Go to vercel.com → New Project → Import repo
3. Set root directory to `web-app`
4. Add env variable: `REACT_APP_API_URL`
5. Deploy! 🚀
